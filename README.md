### ⚡ [1. Product Analytics & Onboarding A/B Testing Case Study](https://github.com/israr1234567890/product-analytics-ab-testing)
*An end-to-end product analysis mapping user churn and evaluating a redesigned interactive tutorial flow.*

*   **The Problem:** Identified a critical drop-off where **50% of new signups churned on Day 1**.
*   **The Analysis:** Designed and evaluated a randomized A/B test (2,120 users) in Python and SQL.
*   **The Findings:** Proved a statistically significant **20.6% absolute conversion lift** ($p < 0.0001$) and **19.9% 7-day retention lift**, while Welch's T-test confirmed no negative impact on Average Order Value (AOV).
*   **Business Impact:** Projected **+$3.10M in annualized revenue lift** at scale.

### 🔍 [2. Customer Segmentation & Churn Analytics (LTV Optimization)](https://github.com/israr1234567890/customer-churn-analytics)
*An end-to-end customer lifecycle analysis using unsupervised and supervised machine learning.*

*   **The Problem:** Subscription platform losing revenue due to customer churn.
*   **The Analysis:** Engineered RFM metrics, segmenting the user base using K-Means Clustering (K=4, PCA 2D viz), and trained a Random Forest Classifier (89.5% accuracy, 0.95 ROC-AUC) to predict churn risk.
*   **The Findings:** Identified a critical "At-Risk Inactive" segment (18.6% of users) exhibiting an **80.5% churn rate**. Proved logins in the last 30 days and inactivity duration are the strongest churn predictors.
*   **Business Impact:** Projected **+$420K in annualized revenue capture** through targeted reactivation triggers.
*   **Deliverables:**
